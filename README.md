## 🚀 הרצה מקומית

אין צורך ב-build — זה אתר סטטי:

```bash
# עם Python
python -m http.server 8080

# או פשוט לפתוח את index.html בדפדפן
```

ואז לגלוש ל-`http://localhost:8080`
